Les fonctions

Utilisation des fonctions

Une fonction est un tout homogène, destiné à remplir une tâche bien définie.

Pourquoi utiliser une fonction?

  • Afin de modulariser un programme(découper un travail complexe en plusieurs tâches simples)
  • Afin d'écrire une tâche qui se repéte plusieurs fois dans un programme.

Une fonction doit :

  • Être déclarée(écriture d'un prototype)
  • Être définie(écriture du corps de la fonction, du programme la composant)
  • être appelée(utilisation de la fonction)

Déclaration d'une fonction ou le prototype de la fonction

Le prototype d'une fonction montre la forme de la fonction : son résultat(le type de la valeur de retour), le nom de la fonction et les paramètres.
Les noms des paramètres est facultatifs, mais leurs types est obligatorires pour le compilateur.

type nom_fonction(type parametre_formel1, type parametre_forme2);

Définition d'une fonction

type nom_fonction(type parametre_formel1, type parametre_forme2) {
    corps de la fonction
}

L'appel d'une fonction

L'appel de fonction dépend des paramètres nécessaires. par exemple, clrscr(); ne nécessite aucun paramètre.

L'instruction return

L'instruction return indique la fin logique de la fonction(fin de son exécution), et le retour à la fonction apellante, elle peut prendre un paramètre de retour ou rien. Retourne toujours qu'une seule valeur.
Trois exemples acceptable d'instruction return :

return valeur_de_retour;
return Var1*Var2/3;
return void;
