New-item, commande de création

Le new-item est une commande de création qui a trois paramètres :

  • -file ou -directoty : indique si on créé un fichier ou un répertoire
  • -path : indique l'endroit où on créé l'item
  • -name : indique le nom de l'item
new-item -path C:\RepX\RepY\Repz -name bob -type Directory
new-item -path C:\RepX\RepY -name bart.txt -type File
New-Item C:\RepX\RepY\marge.txt -Type file

Remove-item, commande de destruction

I have become death, destroyer of world Remove-item C:\RepX\RepY\Repz\bob

Get-content, commande d'acquisition

get-content C:\repx\repy\marge.txt

Set-content et add-content

Set-content écrase le fichier tandis que add-content ajoute aux contenus du fichier.
Si le fichier n'existe pas, ces commandes le crée.

set-content -path C:\repx -name marge.txt -value "you have nothing to loose"
add-content -path C:\repx -name marge.txt -value "but your chains"

Les paramètres génériques

L'étoile peut remplacer des caractères, afin de faire des recherches
Par exemple * ou *b permet de trouver bob
bob.txt ne serait pas trouvé *.txt permettrait de trouver bob.txt

get-childitem C:\RepC12\Rep1\rep2\*

Paramètre -include

Permet de paramètré les informations qu'on veut inclure

get-childitem C:\RepC12\Rep1\rep2\* -include Fich*
get-childitem C:\RepC12\Rep1\rep2\* -include Fi*.txt
get-childitem C:\RepC12\Rep1\rep2\* -include *ic*

Paramètre -exclude

Permet de paramètré les informations qu'on veut


get-childitem C:\RepC12\Rep1\rep2\* -exclude *.txt
get-childitem C:\RepC12\Rep1\rep2\* -exclude Fich*
get-childitem C:\RepC12\Rep1\rep2\* -exclude Fich*
