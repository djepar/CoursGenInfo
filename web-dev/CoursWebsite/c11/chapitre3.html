3.1 La sélection

Pour créer une condtion, il faut créer une instruction de sélection.

  • Règle 1 : La condition menant à la sélection doit être exprimée de manière à n'offrir que deux réponses possibles :vrai ou faux;
  • Règle 2 : si la sélection offre deux possibilités, il faut écrire l'énoncé sous la forme Si-Sinon

3.8 Les fonctions cin.fail() , cin.clear() et cin.ignore()

Voici plusieurs fonctions utilent lorsque l'usager fait un erreur d'entrée de donnée. cin.fail()est une méthode de validation d'entrée de donnée. Cette fonction retourne vrai lorsqu'une erreur est commiste, sinon elle retourne faux.
Une fois que cin.fail() détecte une erreur, le flux cin devient inactif.
De plus, il est important de viter le tampon de lecture à l'aide de cin.ignore().

  Par exemple, pour vider un tampon contenant 80 caractères, il faut écrire : cin.ignore(80,'\n')
  Cela effacera tous les caractères jusqu'au premier '\n' pour un maximum de 80 caractères.

3.9 La sélection multiple et les if()-else imbriqués

Lorsque nous avons à choisir un option sur un lot de plusieurs possibilités, il faut n-1, où n représente le nombre de possibilité (p. 90)

3.10 L'instruction switch()

L'instruction switch() permet une sélection multiple qui est plus clair que plusieurs if()-else imbriqués.
Le paramètre de switch est l'expression évalué, qui doit être un entier (de type char, int, ...)
Il n'y a qu'une paire d'accolade qui fait tout le bloc switch avec les différentes cases.
Une seule alternative devra être choisi. Il est conseillé de mettre une clause par défault au cas qu'aucune donnée valide n'a été entrée.
L'instruction switch() fonctionne en cascade, ce qui veut dire qu'il faut utiliser break pour éviter que les instructions qui suivent ne soient pas exécuté. Il est également possible de jouer avec ce phénomène de cascade s'il faut accumulé plusieurs instructions.
