Chapitre 4

4.1 La répétition

Pour prendre avantage de la rapidité d'un ordinateur, les instructions de répétitions permettent d'effectuer différent types de boucle afin de répéter une même séquence d'instruction plusieurs fois.
Les instructions de répétition sont le Pour, le Tant Que, et le Faire...Tant Que.

4.2 Les répétitions dans les diagrammes d'action

Règles des instructions de répétitions :
règle 1 Les critères de sélection d'un énoncé de répétion sont : Pour répète un bloc d'instruction un nombre connu et exact de répétion Tant Que répète un bloc d'instruction un nombre inconnu de répétion. C'est le changement d'une condition qui arrête la boucle. Faire...Tant Querepète au moins une fois un bloc d'instruction, puis le répète jusqu'à ce qu'il est un changement à la condition.
règle 2
règle 3 Le double crochet signale le début et la fin du bloc d'instruction à répéter.

4.3 L'énoncé Pour

Un Pour est répété tant que sa condition est vraie, c'est pourquoi il s'agit d'une condition pour poursuivre et non pas d'une condition d'arrêt. Notez que pour arrêter cette répétition, ici la valeur de i devra dépasser 10 (ainsi la condition i<=10 sera fausse). Après le Pour, la valeur de i est donc de 11. (p.117)

4.4 L'instruction for

Execution d'un for(), p. 118

  1. Affectation de la valeur initiale à la variable de contrôle.
  2. Évaluation de la condition pour poursuivre :
    • Si la condition est fausse, on quitte le for() (la répétition se termine ici)
    • Si la condition est vraie, on exéute les instructions du for()(son bloc-d'instruction)
  3. Incrémentation (ou décrémentation) de la variable de contrôle.
  4. Retour à l'étape 2.

4.5 L'énoncé Tant que

Cet énoncé est utile lorsque nous avons pas le nombre exact de fois que la boucle doit être répétée.

4.6 L'instruction while()


La boucle continue tant que la condition n'est pas vraie.

Il s'agit de l'énoncé en C++ pour l'instruction de répétition Tant Que.
syntaxe :> : while (condition pour poursuivre)
 bloc-instructions
exemple :

//Lire un âge valide (de 0 à 120 inclusivement)
cout << "Entrez votre \x83ge: ";
cin >> Age;
while (Age<0 || Age>120) {
cout << "\n\x83ge invalide, recommencez";
cin >> Age; } cout << "\n\nVotre \x82ge est : " <<;

4.7 L'énoncé Faire...Tant que

L'énoncé Faire...Tant que, le bloc d'instruction est executé une première fois avant d'atteindre une condition de répétition.

il s'agit d'une condition pour poursuivre, et non pas d'une condition d'arrêt. (p.138)

4.8 L'instruction do...while()

Il s'agit de l'énoncé en C++ pour l'instruction de répétition Tant Que.
syntaxe :> : do
 bloc-instructions
while (condition pour poursuivre);
exemple:

//Lire un âge valide (de 0 à 120 inclusivement)
cout << "Entrez votre \x83ge: ";
cin >> Age;
do {
cout << "\n\x83ge invalide, recommencez";
cin >> Age; } while(Age<0 || Age>120); cout << "\n\nVotre \x82ge est : " <<; Voici comment s'exécute un do…while():
  1. Exécution du bloc-instructions du do…while()
  2. Évaluation de la condition pour poursuivre :
    • si la condition est fausse, on quitte le do…while() (la répétition se termine)
    • si la condition est vraie, on retourne à l'étape 1..(p.139)
