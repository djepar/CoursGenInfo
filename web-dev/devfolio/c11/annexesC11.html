Conversion de type implicite

Les conversions implicitent se font automatiquement par le compilateur.
Par exemple :

// Working of implicit type-conversion #include
using namespace std;

int main() {
// assigning an int value to num_int
int num_int;

// declaring a double type variable
double num_double = 9.99;

// implicit conversion
// assigning int value to a double variable
num_int = num_double;
// Ici, une variable double
cout << "num_int = " << num_int << endl;
cout << "num_double = " << num_double << endl;

return 0;
}

Output :   num_int = 9
  num_double = 9.99

Il faut se rappeler de la possibilité de perte de donnée lors de la conversion de type.

Conversion de type explicent

Les conversions explicent se font manuellement par le compilateur.
La syntaxe pour une conversion en C : (data_type)expression;
La syntaxe pour une conversion en C++ : data_type(expression);
